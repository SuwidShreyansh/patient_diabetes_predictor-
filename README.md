# Diabetes Prediction using Machine Learning

This project predicts whether a person is **diabetic or not** using basic Machine Learning techniques.

# Features

* Predicts diabetes based on health parameters
* Uses Logistic Regression model
* Takes user input from terminal
* Simple and beginner-friendly

# Concepts Used

* Supervised Learning
* Classification
* Logistic Regression
* Data Splitting (Train/Test)

# Technologies Used

* Python
* Pandas
* Scikit-learn

# Dataset Features

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age
  
# Example Input

2,120,70,20,79,25.5,0.5,30

# Output

Result: Not Diabetic
